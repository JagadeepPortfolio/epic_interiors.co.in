# Epic Interiors — Website Build

**Tier:** Pro (₹12,999)
**Stack:** Next.js 14 + Tailwind CSS 3
**Project ID:** 2026-001-epic-interiors

## Pages
1. Home — Hero, services, stats, portfolio preview, testimonials, process, CTA
2. About — Story, philosophy, team
3. Services — 6 service cards with features
4. Portfolio — Tab-filtered gallery (All/Living/Kitchen/Bedroom/Commercial)
5. Contact — Form → WhatsApp, contact info
6. Book Consultation — WhatsApp redirect with benefits
7. Find Us — Google Maps embed, studio details

## Local Development
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push feature branch to GitHub
2. Connect repo to Vercel dashboard
3. Deploy — Vercel auto-deploys on push
4. Set custom domain in Vercel settings (if client has one)
